Вариант 1

1. Потенциал Морзе

Потенциал Морзе часто используется для описания ковалентных связей. Формальная запись потенциала следующая:

\[V^{Morse}_{ij}=D_e^{ij}\left(1-e^{-a_{ij}(r_{ij}-r_{ij}^0)}\right)^2\]

Посчитайте атомарные силы, действующие на атомы.

2. Единицы измерения

Рассматривается системы из частиц, связанных гармоническими пружинами с коэффициентом упругости \(K_s\). Массы частиц в г/моль, расстояния в Ангстремах, время - в фемтосекундах, коэффициент упругости \(K_s\) измеряется в ккал/моль/Ангстрем \(^2\). Для того чтобы система единиц была замкнута, вводится множитель \(\alpha\):

\[m\frac{d^2r}{dt^2}=\alpha\nabla V\]

Найти \(\alpha\).

3. Исправить файл топологии

Даны файлы координат и топологии для некоторой молекулярной системы. Также даны файл для топологии элементов системы и конфигурационный файл .mdp. Но при попытке построения файла .tpr, gmx grompp выдает ошибку. Исправьте файл топологии.

4. Поверхностное натяжение

Рассмотрим систему, состоящую из двух фаз. Граница разделения фаз проходит параллельно плоскости X-Y. В этом случае, поверхностное натяжение может быть рассчитано следующим образом [AlejandreJCP95]:

\[\gamma=\frac{L_{X}}{2}\langle P_{ZZ}-\frac{1}{2}\left(P_{XX}+P_{YY}\right)\rangle\]

Вам даны выходные файлы GROMACS.. Рассчитайте поверхностное натяжение.

AlejandreJCP95

Alejandre, J., Tildesley, D. J., & Chapela, G. A. (1995). Molecular dynamics simulation of the orthobaric densities and surface tension of water. J. Chem. Phys., 102(11), 4574–4583.
